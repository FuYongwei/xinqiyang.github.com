Good Github

Good Github

真的感觉Github很赞,之前写Blog之后,rake deploy需要等个几分钟才能更新vanish缓存,现在直接push之后就更新了.
这点带来的体验还是很不一样的.
其实对于大公司来说,能做到的事情尽量去做到,这个准则同样也适合大部分的开发者.
作为创业者来说更是得追求这些,将一个方面做到极致,做到极致之后,你就会成功!

很赞的体验.事情很多很忙,一直没有闲暇的时间来写博客,最近思考了很多的事情.先将生活安定之后,接下来好好的做自己想做的事情吧!
还有必须花点时间把语言搞好下了!太不方便了!
